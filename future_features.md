# Future Features

Ideas for extending this project beyond downloading files.

---

## Structure Inventory / Parsing

- [x] Parse and summarize a PDB file: chain count, residue count, atom count, ligands present
- [x] Detect missing residues or sequence gaps (compare SEQRES vs ATOM records)
- [x] Extract FASTA sequence directly from ATOM coordinates (independent of download)
- [x] List all ligands/small molecules (HETATM records) with residue names and chain

---

## Geometry & Conformation

- [x] **Ramachandran plot** — phi/psi backbone dihedrals via BioPython calc_dihedral; outlier flagging; PNG output
- [x] **B-factor analysis** — per-residue Cα B-factor; ±2σ flagging; profile plot PNG
- [x] **Radius of gyration** — Cα Rg in Å (or all heavy atoms)
- [x] **Secondary structure assignment** — DSSP per-residue per-frame via MDAnalysis; stacked area plot over MD trajectory

---

## Interactions

- [ ] **Disulfide bond detection** — find CYS-CYS pairs within ~2.05 Å S–S distance
- [x] **Salt bridges** — protein–protein and protein–ligand (ARG/LYS/HIS vs ASP/GLU)
- [x] **Hydrogen bond analysis** — heavy-atom geometry with angle filter; no explicit H needed
- [x] **Pi interactions** — face-to-face, edge-to-face stacking + cation-pi (ARG/LYS/HIS)
- [ ] **Hydrophobic core** — identify buried nonpolar residues
- [x] **Ligand contacts** — list all residues within N Å of each ligand (default 4–5 Å)

---

## Sequence Analysis

- [x] **Sequence conservation mapping** — per-residue Shannon entropy conservation score from a user-supplied multi-FASTA; BLOSUM62 pairwise alignment (no BLAST); outputs bar plot PNG, B-factor PDB for PyMOL colouring, CSV (`conservation.py`)

---

## Multi-structure / Comparative

- [x] **RMSD** — Cα RMSD between two structures; auto sequence alignment; resnum fallback
- [x] **Structure superposition** — superpose mobile onto reference, save transformed PDB
- [x] **Pairwise RMSD matrix** — all-vs-all for N structures; heatmap PNG + CSV export
- [x] **Per-residue RMSD** — per-residue Cα distance after superposition; bar plot PNG
- [x] **NMR ensemble analysis** — per-model RMSD vs reference model; bar plot PNG

---

## Validation / QC

- [ ] **Steric clash detection** — flag atom pairs closer than sum of van der Waals radii
- [x] **Header metadata extraction** — resolution, R-factor, deposition date, organism from REMARK/HEADER
- [x] **Completeness report** — SEQRES vs ATOM residue count, % modeled per chain

---

## Extraction / Conversion

- [ ] **Extract chains** — write each chain to a separate PDB file
- [ ] **Strip solvent/ligands** — output protein-only or nucleic-acid-only PDB
- [ ] **Extract ligand** — write a specific HETATM ligand to its own file
- [ ] **PDB → CIF conversion** — reformat locally without re-downloading

## Visualization

- [x] **Interactive 3D HTML viewer** — py3Dmol, color-coded by interaction type, dashed lines for H-bonds/pi/salt, opens in any browser
- [x] **2D interaction fingerprint** — matplotlib dot-plot (residues × interaction types), PNG output
- [x] **PyMOL .pml script generation** — session script loading structure + XTC trajectory, SS colouring, RMSF B-factor colouring, movie setup; generated by run_md.py
- [ ] **Per-interaction-type separate images** — individual HTML/PNG for each type (contacts-only, hbonds-only, etc.)
- [ ] **LIGPLOT-style 2D diagram** — schematic ligand + surrounding residues (needs RDKit for ligand 2D depiction)

---

## Molecular Dynamics

- [x] **Full MD pipeline** — GROMACS-based: PDB prep (PDBFixer fills missing side-chain atoms) → topology → solvate → minimise → NVT/NPT equil → production run → analysis → plots + PyMOL session (`run_md.py`); tested on 6D1Y at 1 ns
- [x] **MD trajectory analysis** — RMSD, per-residue RMSF, radius of gyration, energy/temperature, H-bond count; DSSP via MDAnalysis (optional, `pip install MDAnalysis`)
- [x] **MD trajectory visualisation** — matplotlib multi-panel summary + individual PNGs; PyMOL .pml loading XTC as states with `mset 1 -N` for correct frame-by-frame playback; RMSF + PC1 B-factor colouring
- [x] **Automated HTML report** — self-contained single-file report with base64-embedded plots, statistics cards (RMSD, Rg, H-bonds, PCA variance), simulation parameter table, all analysis sections, and PyMOL usage instructions
- [ ] **Ligand MD prep** — ACPYPE/ATB integration for small-molecule force-field parameters; merge ligand topology with protein topology
- [ ] **Free energy perturbation** — alchemical FEP setup for relative binding affinity calculations
- [x] **Principal component analysis** — PCA of backbone trajectory via MDAnalysis; scree plot, PC1 vs PC2 phase-space scatter (coloured by time), per-residue PC1 eigenvector magnitude; PC1 B-factor PDB for PyMOL colouring
- [ ] **Binding site volume** — track pocket volume over MD trajectory (e.g., via fpocket or MDAnalysis)
